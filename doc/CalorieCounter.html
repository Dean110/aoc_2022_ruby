Overview

CalorieCounter is for the first day of Advent of Code 2022. The problem was to read a list of integers, determine which elf represented in that list had the most calories, and return the sum of their calories.

def highest_calorie_count
  elves.map { |elf| elf.split("\n").map(&:to_i).sum }.max
end
